# Chocolate Sales Data Analyzer

A Python-based **Chocolate Sales Data Analyzer** that allows users to explore, clean, analyze, search, and visualize chocolate sales data using **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn**.

This project is completely menu-driven and designed for beginners learning Python data analysis and visualization.

## Features

### Data Exploration
- View first and last 5 records
- Display dataset columns
- Check data types
- View dataset information
- Generate statistical summary
- Display dataset shape

### Data Cleaning
- Display missing values
- Fill numeric missing values with mean
- Forward fill missing values
- Drop missing rows
- Replace missing values with custom values

###  NumPy Operations
- Convert columns into NumPy arrays
- Array indexing and slicing
- Element-wise operations
- Square and square root
- Statistical calculations
- Percentiles
- Mean, Sum, Variance, Standard Deviation

### Mathematical Operations
- Column-wise Sum
- Mean
- Maximum
- Minimum
- Product
- Cumulative Sum
- Absolute Values

### Search, Sort & Filter
- Search Product
- Search Sales ID
- Search Sales Amount
- Sort data by any column
- Filter product records
- Aggregate functions
- Statistical analysis
- GroupBy analysis
- Sales normalization

### Data Visualization
- Bar Chart
- Line Chart
- Scatter Plot
- Pie Chart
- Histogram
- Stack Plot
- Subplots
- Heatmap
- Box Plot
- Save generated plots

# Installation

Clone the repository

bash
git clone https://github.com/your-username/chocolate-sales-analyzer.git


Move into the project directory

bash
cd chocolate-sales-analyzer


Install required libraries

bash
pip install pandas numpy matplotlib seaborn

#  Run the Project

bash
